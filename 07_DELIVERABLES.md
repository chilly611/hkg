## Monthly operating deliverables

### Every month, we publish or ship
- **State of the Gardens** — short public post, one per garden, updating users on what shipped, what changed, what's coming. Builds trust and AI-citation inventory.
- **Changelog** — developer-readable changelog for the Machine Lane API and MCP server.
- **Content cadence** — at least 2 Bloom Ledger entries / week (30–60s) + 1 long-form essay / month.
- **Data freshness report** — public page showing when each data source last updated. Trust-building and SEO/AI-citation value.

### Every month, we review internally
- **Pattern integrity review** — has anything we shipped weakened the four-lane / three-surface pattern?
- **Gravity Well metrics** — monthly visitors, tool usage, share rate, AI-citation count
- **MRR dashboard** — per-garden, per-lane, per-surface revenue
- **Lessons log review** — any repeat mistakes? Update `09_LESSONS.md`
