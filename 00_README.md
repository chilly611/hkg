## What this folder is

This is the **umbrella-level** workspace for the Knowledge Gardens company — the parent above HKG (Healthcare), BKG (Builders), OKG (Orchids/Botanical), and the 40+ gardens on the frontier map. Individual garden builds live in their own project workspaces. This folder holds the **forest-level** thinking: strategy, branding, platform architecture, product catalog, research, and the living operating docs (lessons, todos, deliverables).

## Operating protocol

1. **Every session starts here.** Read `00_README.md`, then `09_LESSONS.md`, then `08_TODO.md`.
2. **Strategic decisions update `01_STRATEGY.md`.** Don't let strategy drift silently.
3. **After any user correction**, append to `09_LESSONS.md`. Never repeat a mistake twice.
4. **One deliverable per session** unless explicitly multi-part. Ship it, log it, update state.
5. **The pattern is sacred.** Four lanes, three surfaces, one graph, RSI heartbeat. If a new idea weakens the pattern, the pattern wins.
