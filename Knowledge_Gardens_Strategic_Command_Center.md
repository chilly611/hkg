## The Architecture — One Pattern, Any Industry

Every Knowledge Garden deploys the same proven architecture:

**Four Lanes** serving every participant in an industry:

| Lane | Who It Serves | What It Does | Revenue Model |
|------|--------------|--------------|---------------|
| **Administrator** | Hospitals, GCs, firms, agencies | Credentialing, compliance, operations | Enterprise SaaS, verification API |
| **Professional** | Doctors, contractors, attorneys, advisors | Intelligence, career, certification | Subscriptions, marketplace fees |
| **Public** | Patients, homeowners, consumers | Free gravity well — pulls everyone in | Free → premium conversion, affiliate |
| **Machine** | AI agents, EHR vendors, startups | API, MCP server, certified data feeds | Usage-based API tiers, enterprise feeds |

**Three Surfaces** matching different user contexts:

- **Gold (Dream Machine)** — Aspirational, concierge feel. Emotional, inviting, trust-building.
- **Green (Knowledge Garden)** — Scientific, verified, educational. Evidence-based, authoritative.
- **Red (Killer App)** — Operational, urgent, metric-driven. Speed, accuracy, compliance.

**Core Technology:**

- Neo4j Knowledge Graph — the brain. Relationships are the product.
- Supabase PostgreSQL — transactional backbone (users, billing, sessions)
- Claude AI — intelligence layer (Morning Briefings, 3-Point Verification, NLP queries)
- MCP (Model Context Protocol) — makes every garden natively accessible to any AI agent
- RSI Heartbeat — recursive self-improvement. Auto-ingests updates daily. Gets smarter over time.
