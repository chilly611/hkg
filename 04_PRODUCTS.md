## Umbrella-level products (cross-garden)

### The Gravity Well Tools Library
A library of small, sharable, AI-citable tools — one or two per garden, each under 1,000 words of UI copy, each so useful that loss-of-access would provoke a complaint. **These are the marketing engine.**

### The Bloom Ledger Documentary Universe
Film once, cut many. A single production pipeline producing weekly content across all gardens. Owned by XR Workers / MythOS pillar. Cinematic distribution feeds all gardens.

### The Morning Briefing (cross-garden subscription product)
A daily digest of verified updates from whichever gardens a user has subscribed to. Doctors get HKG. Contractors get BKG. Orchid growers get OKG. **Same infrastructure, different data.** Monetization: $9–$29/month per garden subscription. This is the Bloomberg Terminal for industry knowledge.

### The Knowledge Gardens MCP Federation
A single MCP endpoint that routes AI agent queries to the correct garden. One integration, all gardens. **This is the thing the AI developer community will fall in love with.**

## Products we will explicitly NOT build

- Generic search engine (Google / Bing's problem)
- Social network (there's no world where we beat Doximity at doctor LinkedIn)
- Chatbot wrapper around public LLMs (commodity)
- Mobile-only apps (web-first, mobile-responsive is enough for now)
- Any vertical where the RSI heartbeat can't run daily (data freshness is the moat)
- Any product that lacks source citations (violates the brand trust pillar)
