## Regulatory Positioning: Regulation Is Our Moat

**FDA CDS Guidance (Jan 6, 2026):** Loosened oversight for AI clinical decision support. Our 3-Point Verification system fits exactly within expanded enforcement discretion — we augment clinical judgment, not replace it. Commissioner Makary: "trust clinicians to do what's right."

**Anti-Kickback Statute:** Seventh Circuit (Sorensen, April 2025) held advertising/marketing payments are not illegal referrals. Second Circuit affirmed Zocdoc's booking fee model as AKS-compliant. Cash-pay longevity services carry zero AKS exposure.

**Our 4-lane architecture is a regulatory hedge by design:** Provider-facing tools (Doctor Lane) benefit from CDS guidance. Patient tools (Gravity Well) stay in general wellness. Insurer products (Admin Lane) sell mandatory compliance. Machine Lane serves data, not clinical decisions. Each lane fits within existing safe harbors.

The platform autonomously ingests updates daily: new clinical trials, PubMed publications, NPI changes, OIG exclusions, FDA approvals, drug interactions. Every day the system runs, it gets smarter. Twelve months in, the compounding intelligence gap becomes mathematically insurmountable for competitors who update quarterly or annually.

### 3. AI Citability Engine

Every entity page serves structured JSON-LD schema markup, llms.txt, and Citation Capsules — 40-80 word standalone paragraphs optimized for AI extraction. When any AI assistant globally answers a healthcare question, our architecture makes us the algorithmic favorite for citation. Pages with clean structure and schema earn 2.8x higher AI citation rates. Zero marginal distribution cost.

### 4. MCP Server — The "USB-C for AI"

Our Model Context Protocol server lets any AI agent — diagnostic bots, surgical robots, EHR vendors — query the Knowledge Graph directly with complex relational questions. This is not a feature. It is the infrastructure layer for the AI-driven medical economy.

### 5. Network Effects — Cross-Lane, Cross-Vertical

Every administrator who credentials a doctor improves the data for patients. Every patient interaction refines the knowledge for doctors. Every machine integration validates and extends the data layer. And every vertical shares the same Machine Lane API — healthcare provider data feeds insurance verification, pharma clinical trial data feeds doctor intelligence, construction licensing data feeds real estate agent verification.
